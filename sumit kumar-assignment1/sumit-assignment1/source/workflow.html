Home Page
Customers can go to four different pages (Shop, Feedback, About, Sign In) from home page.
Vertical page options would be given on vertical device.
Sign In Page
Customer has to first sign in with EasifyShop before shopping.
Email verification would be done before signing in the user (email must contain ‘@’ and ‘.’ as characters).
Shop Page
Sign In would be must before adding the product to the cart.
User can add displayed product to his cart. User can choose quantity of a particular product before adding it to the cart.
Then, the user can place order on clicking place order button after adding the products to the cart.
Order Detail Page
The products added to the cart will be shown in tabular format on this page.
After adding the shipping address, the user can proceed to pay on clicking the pay button and window alert will confirm that order is placed.
Shipping address verification will also be done (It should not be empty).
Feedback Page
Sign In would be must before submitting the feedback form.
Email address verification would be done (email must contain ‘@’ and ‘.’ as characters).
Email Address should match with the signed in email address.
